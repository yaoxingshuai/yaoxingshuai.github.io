hexo 使用

1. 看样子我是安装了 hexo 3 版本，网上很多教程是 hexo 2 版本的教程，所以有些指令可能用不了。
2. hexo 可以部署在自己的服务器上， 只需要
3. 为了方便，也可以吧hexo的网页托管到github，这样就不需要自己单独开启一个服务端了，github会自动渲染hexo文件。
  • 首先你需要在github上面创建一个项目。比如你的github账户是 abc，那么项目名 必须 是 abc.github.io.git
4. 需要用到的指令

hexo deploy 可以自动推送到github，但是因为版本问题，这个指令不太好用，所以我就 直接把 hexo/pulic 的内容推送到 git@github.com:yaoxingshuai/yaoxingshuai.github.io.git
